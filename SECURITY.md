# Security Policy

## Reporting a Vulnerability

This is a personal GitHub profile repository. It does not contain application code or services.

However, if you discover a security concern related to:

- **GitHub Actions workflows** (e.g., token leakage, injection vulnerabilities)
- **External service misconfigurations** (e.g., exposed secrets in SVGs or badges)
- **Supply chain risks** (e.g., compromised action dependencies)

Please report it responsibly by emailing: **manavag70@gmail.com**

I will acknowledge the report within 48 hours and work with you to address the issue.

## Scope

| In Scope | Out of Scope |
|----------|-------------|
| GitHub Actions workflows | Third-party badge services (shields.io, etc.) |
| Repository secrets management | External API endpoints |
| SVG/asset injection risks | Linked project repositories |

## Supported Versions

| Version | Supported |
|---------|-----------|
| `main` branch | ✅ |
| `output` branch | ✅ (auto-generated) |
