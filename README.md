### 🔊 &nbsp;PixelSonic

> **Offline image & text transmission via audio** — encodes arbitrary data into acoustic signals using FSK/PSK modulation, enabling file transfer with zero network infrastructure. A software-defined modem built in Python.

**Architecture**

```
Data → Encoder
         ├── FSK Modulation (bit → freq mapping)
         ├── PSK Phase Shifting
         └── WAV Audio Stream
                  ↓
           Microphone Capture
                  ↓
         Decoder → FFT Analysis
         → Demodulation → Data
```

**Key Features**
- **FSK + PSK** hybrid modulation scheme
- **FFT-based** spectral analysis for demodulation
- Full image reconstruction from raw audio
- Works completely **offline** — zero Wi-Fi required
- Sub-100ms latency for text payloads

### 🌐 &nbsp;Tranzcendia

> **Universal accessibility platform** that demolishes language and sensory barriers — combining real-time speech translation, multilingual text translation, and Braille output in a single, accessible app.

**Architecture**

```
Audio Input → Whisper (STT)
                  ↓
         Translation Engine
         (MarianMT / DeepL API)
                  ↓
     ┌────────────┴────────────┐
  Text Output           Braille Renderer
  (50+ languages)       (BRF format)
     └────────────┬────────────┘
               FastAPI
            (REST + WebSocket)
```

**Key Features**
- Real-time **speech-to-text** in 50+ languages
- **Braille (BRF)** output for screen readers
- Offline text translation mode with caching
- **WCAG 2.1 AA** accessibility compliance
- Cross-platform: Web + Android (Kotlin)
